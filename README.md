# GURUSUPO
 
GURUSUPOとは現在地から周辺の飲食店を探し出すWebサービス
 
## 主な機能
・現在地からいますぐ検索（半径500ｍ）
・条件検索（距離、カード利用、電子マネー利用、駐車場、WIFI）
・グーグルマップにマーカーで表示
・お気に入りに登録することでページを離れるまでアイコンの色とマップ上のマーカーの色を変えることが出来る

## 使用したフレームワーク、ライブラリ、
・Vue.js(jQuery未使用)
・vue-paginate
・vue-js-modal

## メモ
